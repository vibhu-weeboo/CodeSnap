## ğŸ¨ Usage Guide

### Basic Usage

1. **Paste Your Code**: Copy and paste your code into the editor
2. **Select Language**: Choose from 100+ supported languages
3. **Choose Theme**: Pick from popular themes like Dracula, Monokai, Nord
4. **Customize**: Adjust padding, shadows, window style, and more
5. **Export**: Download as PNG/SVG or copy to clipboard

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+V` | Paste code |
| `Ctrl+C` | Copy image to clipboard |
| `Ctrl+S` | Download image |
| `Ctrl+K` | Clear editor |
| `Ctrl+T` | Change theme |
